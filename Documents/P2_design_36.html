Purpose goals and audience

The purpose is still to make a website that people can visit to buy tickets to the cinema, and also get information about the movies shown in the cinema. The website should be easy to access for everybody, as the audience for the website is a very broad group of people. In order to achieve this the website has to be intuitive as well as having a good layout that also makes the visitors of the website feel like it is an easy-to-understand website.

Navigation Structure

The navigation structure is a clique where every page is connected to every other page. The reason for this is the menu at the top of each page, which is the same no matter which page you are on. The user can access any of the pages through this menu at all time.
